📘 My Academic Journey

Here are some of the important courses I have taken during my undergraduate studies:

🧑‍🎓 Semester 1

  • Computer Applications: 💻 An introduction to Computers, Operating Systems, Networking, Security, Presentation, Word Processing, and Spreadsheet Software.
  • Mathematical Methods I: 📈 Fundamentals of Algebra, Functions & Graphs, Exponential & Logarithm Functions, Trigonometry, Vectors, Limits, Derivatives, etc.

🧑‍🎓 Semester 2

  • Mathematical Methods II: 🔢 Integration, Differential Equations, Infinite Sequences & Series.
  • Principles of Programming: 🖥️ Programming Concepts, Variables, Loops, Functions, File Handling, Recursion, Sorting, etc.
  • Probability & Statistics I: 🎲 Set Theory, Probability, Random Variables, Sampling, Hypothesis Testing, etc.

🧑‍🎓 Semester 3

  • Management Information Systems: 📊 Information Systems, ERP, CRM, SCM, Decision Support Systems, etc.

  • Probability & Statistics II: 📊 ANOVA, Chi-squared Tests, Regression, Time Series (Excel & R Programming).

🧑‍🎓 Semester 4

  • Introduction to Econometrics: 📉 OLS, Multicollinearity, Autocorrelation, Binary Dependent Variables, Panel Data, Time Series. R Project for Module
  • Data Visualization: 📊 PowerBI, DAX, Tableau. Power BI Project
  • Operations Research I: 🔄 Linear Programming, Network Optimization, Dynamic Programming, Integer Programming, etc.
  • Systems Analysis & Design: ⚙️ Object-Oriented Systems, Project Management, Use Case Diagrams, Requirement Gathering, etc.
  • Seminar IV: 🛠️ Introduction to Machine Learning, No-Code Platforms (Workhub24), MS PowerApps. Power Automate Project

🧑‍🎓 Semester 5

  • Database Management: 🗂️ Relational Models, SQL, ER Diagrams, Normalization, Storage & Indexing, etc.

🧑‍🎓 Semester 6

  • Business Application Development: 🌐 Web Development (HTML, CSS, Django), Git, No-Code Development, etc. Django Project, Git Repo
  • Spatial Data Analytics: 🌍 GeoDa, QGIS, Google Earth Engine. Spatial Data Project
  • Data Mining: ⛏️ Association Rule Mining, Classification, Clustering, Data Warehousing.
  • ERP Systems: 🔗 Life Cycle, Implementation, Integration, Customization, etc.

🧑‍🎓 Semester 7

  • Text Analytics: 📝 Linguistics, Regular Expressions, Deep Learning for NLP. Project, Git Repo

  • R&D Project: 🔬 "Improving Knowledge Management Systems (KMS) with LLMs."

🧑‍🎓 Semester 8
