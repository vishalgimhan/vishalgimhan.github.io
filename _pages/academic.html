Here are some of the important courses I have taken during my undergraduate studies:

🧑‍🎓 Semester 1

  • Computer Applications: 💻 An introduction to Computers, Operating Systems, Concepts of Hardware & Software, Networking, Security, Data & Information, Presentation Software, Word Processing, and Spreadsheet Software.
  • Mathematical Methods I: 📈 Fundamentals of Algebra, Functions & Graphs, Exponential & Lograithm Functions, Trigonometry, Vectors, Limits, Derivatives, etc.

🧑‍🎓 Semester 2

  • Mathematical Methods II: 🔢 Introduction to Integration, Differential Equations, Infinite Sequences & Series..
  • Principles of Programming: 🖥️ Concepts of Programming, Elements(Expressions, Variables, Operators), Conditions, Loops, Functions, Sequences, File Handling, Classes, Recursion, Sorting, etc.
  • Probability & Statistics I: 🎲 Set Theory, Probability, Random Variables, Probability Distributions, Descriptive Statistics, Sampling, Confidence Intervals, Hypothesis Testing, etc.

🧑‍🎓 Semester 3

  • Management Information Systems: 📊 Information Systems, Information Technology, Business Applications (ERP, CRM, SCM, etc), Decision Support Systems, etc.
  • Mathematical Methods III: 🔍 Calculus, Systems of Linear Equations, Matrices, Introduction to Jupyter Notebooks, Eigenvalues & Eigenvectors.
  • Probability & Statistics II: 📊 Two-sample tests, ANOVA, Chi-squared tests, Non-parametric tests, Linear Regression, Multi Linear Regression, Dealing with Categorical variables, Introduction to Time Series, Practiced through Excel & R programming language, etc.

🧑‍🎓 Semester 4

  • Introduction to Econometrics: 📉 Properties OLS, Multicollinearity, Heteroskedasticity, Autocorrelation, Model Specification Error, Data related Errors (Measurement errors, Missing, Outliers), Binary Dependent Variables, Panel Data, Time Series, Serial Correlation, etc. R Project for Module
  • Data Visualization: 📊 PowerBI, DAX, Tableau. Power BI Project
  • Operations Research I: 🔄 Linear Programming, Network Optimization, Dynamic Programming, Integer Programming, etc.
  • Systems Analysis & Design: ⚙️ Object-Oriented Systems, Project Management, Use Case Diagrams, Requirement Gathering, etc.
  • Seminar IV: 🛠️ Introduction to Machine Learning, MS Azure, No Code-Low Code Platforms (Workhub24), Business Process Automation using MS PowerApps. Power Automate Project

🧑‍🎓 Semester 5

  • Database Management: 🗂️ Relational Models, Introduction to SQL, ER Diagrams, Relational Mapping, Normalization, Storage & Indexing, etc.
  • Data Structures & Algorithms: ⚙️ Data Structures (Arrays, Linked Lists, Stacks, Queues, Trees, Graphs, Hash Tables, etc), Algorithms (Sorting, Searching, Greedy, Dynamic Programming, Divide & Conquer, etc)
  • Data Wrangling: 🧹 Data Cleaning, Preprocessing, Standardization, Integration, Transformation, Reduction, Generalization and Summarization, etc
  • Multivariate Methods: 🔬 Multivariate Normal Distribution, Principal Component Analysis, Factor Analysis, Multivariate Linear Regression, etc.
  • Stochastic Processes: 📈 Univariate Distributions, Moment Generating functions, Maximum Likehood Estimation, Goodness of fit, Queuing Theory, Random Walks, Markov Chains, Poisson Processes, etc.

🧑‍🎓 Semester 6

  • Business Application Development: 🌐 IDEs, Debugging, Web Applications, HTML, CSS, Django, Version Control, Git, Web APIs, Deployment, Web Services, No-Code development, Application Testing. Django Project, Git Repo
  • Spatial Data Analytics: 🌍 Spatial Data, GeoDa Application, Spatial Autocorrelation, QGIS application, Google Earth Engine. Spatial Data Project
  • Data Mining: ⛏️ Association Rule Mining, Prediction, Classification, Clustering, Data Pre-processing, Data Warehousing & Modelling, Data Cube, etc.
  • ERP Systems: 🔗 Life Cycle, Implementation, Integration, Customization, etc.
  • Numerical Methods: ➗ Linear Systems, Numerical Representation, Significant Digits, Non-linear Equations(Bisectionm Newton's, Secant), Interpolation, Numerical Differentiation & Integration, Ordinary Differential Equations, Runge Kutta, etc.
  • Operations Research II: 🎯 Analytical Hierarchy Process, System Dynamics, Decision Analysis, Dynamic Programming, Evolutionary Algorithms, Non-Linear Programming, LP with Recourse, etc.

🧑‍🎓 Semester 7

  • Machine Learning: 🤖 Statistical Learning, Linear Regression, Model Selection Methods, Clustering (K-Means & Hierarchical), Classification (K-Nearest Neighbors, Logistic, Support Vector Machines), Decision Trees, Classification Metrics, Bagging, Boosting, Random Forest, Cross Validation, etc. ML Project for Module
  • Deep Learning: 🧠 Multi-Layer Perceptron, Convolutional Neural Networks, Deep Generative Models, Deep Sequence Models, Transformers, Multi-Task Learning, Transfer Learning, Meta Learning, Reinforcement Learning, Large Language Models, etc. Project, Git Repo
  • Text Analytics: 📝 Basic Linguistics, Regular Expressions, Text Preprocessing, Semantics, Deep Learning for NLP, etc. Project, Git Repo
  • Financial Econometrics (Time Series): 💰 Stationarity, Box Jenkin Methodology, Simultaneous Equation Systems, VAR, Cointegration, Volatility, ARIMA, GARCH, etc.
  • R&D Project: 🔬 Undergraduate Research in “Improving Knowledge Management Systems (KMS) with Large Language Models (LLMs)”

🧑‍🎓 Semester 8

  • Advanced Database Management: 🗄️ Relational Algebra, Query Processing, Query Optimization, Transactions, Concurrency, Distributed Databases, NoSQL, MongoDB, Information Retrieval, Blockchain Databases, etc.
  • Business Intelligence: 📊 Big Data (Distributed Processing, MapReduce, Hadoop, Big Data EcoSystem, Big Data Architecture), Business Intelligence (Data Warehousing, Data Lakes, Data Vault, Data Mesh, PowerBI, Tableau). BI Project, Git Repo
